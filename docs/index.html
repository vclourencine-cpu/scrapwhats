Extraia contatos dos seus grupos do WhatsApp

Ferramenta gratuita que roda diretamente no seu navegador. Nenhum dado sai do seu computador.

Como usar — passo a passo

1

Mostre a barra de favoritos do seu navegador

Antes de tudo, a barra de favoritos precisa estar visível na tela.

Google Chrome / Edge: Pressione Ctrl + Shift + B
Firefox: Pressione Ctrl + Shift + B

Uma barra vai aparecer abaixo da barra de endereço com os seus favoritos.

2

Arraste o botão abaixo para a barra de favoritos

Clique e segure o botão azul abaixo, arraste até a barra de favoritos e solte. Vai aparecer como um novo favorito.

Clique, segure e arraste para a barra acima
⚡ ScrapWhats
Segure e arraste ↑ para a barra de favoritos
3

Acesse o WhatsApp Web e faça login

Abra uma nova aba e acesse web.whatsapp.com.

Escaneie o QR Code com seu celular:
WhatsApp no celular → Configurações → Aparelhos conectados → Conectar um aparelho

Aguarde a lista de conversas carregar completamente.

4

Clique em ⚡ ScrapWhats na barra de favoritos

Com o WhatsApp Web aberto, clique no favorito "⚡ ScrapWhats" que você acabou de adicionar.

Um painel vai aparecer na tela. Selecione os grupos que deseja extrair e clique em "Extrair Contatos".

O arquivo Excel será baixado automaticamente.

Se não conseguir arrastar, siga estes passos alternativos:

1. Abra o WhatsApp Web e faça login.

2. Pressione F12 para abrir o painel de desenvolvedor.

3. Clique na aba "Console".

4. Copie o código abaixo e cole no Console, depois pressione Enter:

var s=document.createElement('script');s.src='https://vclourencine-cpu.github.io/scrapwhats/bookmarklet.js?v='+Date.now();document.head.appendChild(s);

* Se o console pedir para digitar "allow pasting", escreva isso e pressione Enter, depois cole o código novamente.
